*What the heck is a pentester?

A pentester, short for penetration tester, is a security professional.

He is responsible for evaluating the security of an organization's systems and the network with the associated applications.

The goal of a pentester is to identify vulnerabilities in a system and then report them to the organization so that they can be fixed before malicious actors can exploit them.

(also known as white hat hackers)
